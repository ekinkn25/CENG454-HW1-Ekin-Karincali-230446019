# CENG 454 - HW1: Sky-High Prototype

This repository contains the first homework assignment for the CENG 454: Game Programming course. 

The project demonstrates a basic aeronautical prototype built with Unity 6. 
It includes a high-resolution 3D aircraft model managed via Git LFS and a C# script.

## Flight Controls
* Pitch: Up Arrow key/W and Down Arrow Key/S
* Yaw: Left Arrow Key/A and Right Arrow Key/D
* Roll: Q and E Keys
* Forward Thrust: Spacebar

## Technologies Used
* Unity 6
* C#
* Git & GitHub  & Git LFS
